## City Bike
City bike is a mobile appication where users can search for the nearest bike rental stores and also search for good spots to bike around.

## Technologies 


| Technologies  | APIs            | 
| ------------- |:-------------:  | 
| React Native  | Google Maps API |
| CSS           | Foursquare AP   | 
| Javascript    |
| Bootstrap     |
| AXIOS         |
